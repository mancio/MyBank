# MyBank 
## The rest API money transfer app

#### Description

This is a webapp with an online user panel interface to manage bank accounts and transfer money from an account to
another.

#### Design

* Font-end: AngularJs
* Back-end: Java + Spring + oauth2 (security Log in)

#### RestAPI

* [http://localhost:8080//
